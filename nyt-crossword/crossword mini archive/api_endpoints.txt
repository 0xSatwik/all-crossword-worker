NYT Mini Crossword Archive API
================================

Base URL
- Production: https://nyt-mini-archive.nytsolver.workers.dev

Auth
- Read endpoints are public.
- Write endpoints require `Authorization: Bearer <API_KEY>`.
- Legacy path-token routes such as `/today/add/<key>` and `/date/add/<key>` return `410 Gone`.

Public Endpoints

1. GET /
- Purpose: root docs and quick health check.
- Response: JSON with the available endpoints.

2. GET /today
- Purpose: fetch the latest stored mini puzzle.
- Response: puzzle data for the latest available stored date.

3. GET /date?date=YYYY-MM-DD
- Purpose: fetch one stored puzzle by date.
- Required query:
- `date`: exact date like `2026-05-30`

4. GET /formatted
- Purpose: fetch formatted crossword text for the latest stored puzzle.

5. GET /formatted?date=YYYY-MM-DD
- Purpose: fetch formatted crossword text for one stored date.

6. GET /clue?q=TEXT&mode=exact|contains
- Purpose: search stored mini clues.
- Required query:
- `q`: clue text
- Optional query:
- `mode`: `exact` or `contains`

7. GET /solve?clue=TEXT&pattern=OPTIONAL
- Purpose: solver-style clue lookup from the stored mini archive.
- Required query:
- `clue`: clue text
- Optional query:
- `pattern`: answer pattern using `?` wildcards, for example `C??`
- Response shape:
- `answers`: ranked candidate answers
- `history`: matching stored clue rows

8. GET /answer?q=TEXT&mode=exact|contains
- Purpose: search stored answers.
- Required query:
- `q`: answer text
- Optional query:
- `mode`: `exact` or `contains`

9. GET /list?page=N&limit=N
- Purpose: list stored puzzle dates with pagination.
- Optional query:
- `page`: defaults to `1`
- `limit`: defaults to `50`, max `100`

Protected Endpoints

10. POST /today/add
- Purpose: fetch the latest NYT mini puzzle and store it.
- Required header:
- `Authorization: Bearer <API_KEY>`

11. POST /date/add?date=YYYY-MM-DD
- Purpose: fetch and store one specific NYT mini puzzle by date.
- Required header:
- `Authorization: Bearer <API_KEY>`
- Required query:
- `date`: exact date like `2026-05-30`

Common Status Codes
- `200`: request succeeded
- `400`: missing or invalid query value
- `401`: missing or invalid bearer token on write routes
- `404`: unknown route
- `405`: wrong HTTP method on write routes
- `410`: legacy path-token write route removed

Examples
- Read latest: `GET https://nyt-mini-archive.nytsolver.workers.dev/today`
- Solve clue: `GET https://nyt-mini-archive.nytsolver.workers.dev/solve?clue=Cookie+brand`
- Add latest:
  `POST https://nyt-mini-archive.nytsolver.workers.dev/today/add`
  Header: `Authorization: Bearer <API_KEY>`
